# unite-roulette 🎰

ポケモンユナイトの **チーム編成をランダム抽選** するWebアプリＵ。
人数を選んで抽選すると、ポケモンと技構成（わざ1・わざ2）がランダムに決まります。

- **状態**: 🛠 MVP（動作する最小版）
- **種別**: Web（React + TypeScript + Vite）

## できること

- **人数を1〜5人で選択**して一括抽選（チーム内でポケモンは重複しない）
- **枠ごとにロール指定**（おまかせ／アタック／ディフェンス／バランス／スピード／サポート）
- 抽選時は **名前がパラパラ切り替わって1人ずつ確定する演出**
- **編集画面**でポケモン・技を追加／修正／削除（コードを触らず更新できる）
  - 変更は**ブラウザに自動保存**され、リロードしても残る
  - **JSON書き出し／読み込み**でバックアップ・共有・アプデ反映
  - **初期データに戻す**でリセット

## 使い方（開発）

```bash
npm install     # 初回のみ：依存をインストール
npm run dev     # 開発サーバー起動（http://localhost:5173）
npm run build   # 本番用にビルド（dist/ が生成される）
npm run preview # ビルド結果をローカルで確認
```

型チェック（TypeScript）:

```bash
npx tsc -b
```

## 構成

```
src/
├─ data/
│  ├─ types.ts     … データの「型」定義（Pokemon / Role など）
│  └─ pokemon.ts   … 初期ポケモンデータ（SEED_POKEMON）
├─ lib/
│  ├─ draw.ts      … 抽選ロジック（純粋関数・画面から独立）
│  └─ storage.ts   … ブラウザ保存・JSON入出力・壊れたデータ対策
├─ components/
│  ├─ Roulette.tsx … 抽選画面＋演出
│  └─ Editor.tsx   … 編集画面
└─ App.tsx         … データを1か所で持ち、タブで画面を出し分け
```
